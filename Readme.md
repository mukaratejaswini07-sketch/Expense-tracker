# Expense Tracker Web Application

A **modern and colorful Expense Tracker** built using **HTML, CSS, and JavaScript**. This project helps users track their income and expenses, calculate balance in real time, and store data persistently using **localStorage**. The UI is designed with a bold, centered card layout and attractive gradients.
